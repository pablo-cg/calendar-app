# React Calendar App

## Modo Desarrollo

1. Renombrar el archivo `.env.example` a `.env`.
2. Cambiar las variables que corresponden.
3. Instalar dependencias

```bash
pnpm install
```

4. Lanzar la aplicacion en modo desarrollo

```bash
pnpm dev
```
